# TrustAI Marketplace

TrustAI Marketplace is an AI-powered marketplace assistant designed to help buyers make safer purchasing decisions by analyzing online listings for authenticity, pricing, and potential scam indicators.

This project is being developed as part of the Quantic Master of Science in Software Engineering (MSSE) Capstone Project.

---

## Project Goals

- Analyze marketplace listings using AI
- Detect potential scams and suspicious listings
- Evaluate price plausibility
- Improve buyer confidence
- Demonstrate modern software engineering practices
